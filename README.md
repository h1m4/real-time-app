# real-time-chat

## Real Time chat

 * Bootstrap, Vue.JS와 Firebase를 통해 만든 채팅 앱
 * Google 로그인 기능 > 로그인 해야 메인 페이지로 들어감



 ## 느낀점 및 반성

 * Firebase를 연동하는 건 처음 해 봤다.
 * Vue.JS에 대해 더 자세히 알아야 겠다...
 * 현재 다른 아이디에서 보낸 것과 내가 쓴 메세지를 구분 못하고 있음


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
